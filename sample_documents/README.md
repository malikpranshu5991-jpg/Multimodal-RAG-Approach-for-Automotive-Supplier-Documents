# Sample Document Notes

This folder contains the public sample PDF used for the project demo.

## Included document

- `continental_supplier_requirements_manual_2024.pdf`

This is the **Continental Supplier Requirements Manual**. It is a strong fit for the project because it includes:

- supplier approval and quality-system prerequisites,
- sustainability and due-diligence requirements,
- logistics, storage, packaging, and transportation requirements,
- product marking, labeling, delivery note, traceability, and inspection certificate rules,
- capability, PFMEA, control plan, and complaint-handling expectations.

## Usage note

For the cleanest demo, delete any old local vector index, start the API, and ingest only this sample file before running queries or taking screenshots.
